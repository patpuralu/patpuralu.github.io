ShockwaveTR

Herramienta OSINT de línea de comandos con 5 módulos independientes: rastreo de IPs con geolocalización completa, información de tu propia red, tracker de números de teléfono, buscador de usuarios en 22 redes sociales en paralelo, y lookup de dominios. Incluye historial en JSON, soporte para daltonismo y banner ASCII de Shockwave.

Activo Python 3 OSINT Threading REST APIs Terminal JSON

Módulos disponibles

  • IP Tracker — geolocaliza una IP: país, ciudad, coordenadas, ISP, ASN, timezone, enlace a Google Maps y hora local
  • Show Your IP — muestra tu IP pública y toda la info de red local: CIDR, broadcast, rango usable, gateway, subredes /24
  • Phone Tracker — analiza un número de teléfono: operadora, región, timezone, formato E.164, validez, tipo (móvil/fijo/VoIP)
  • Username Tracker — busca un username en 22 redes en paralelo. Soporta 1 usuario o varios a la vez
  • Domain Lookup — resuelve un dominio a IP y obtiene país, ISP, registrador, fecha de creación y registros DNS

Tecnologías y librerías

  • requests — peticiones HTTP a las APIs externas
  • phonenumbers — parseo y validación de números de teléfono
  • ThreadPoolExecutor — escaneo paralelo de las 22 plataformas
  • ipaddress — validación y cálculo de redes IPv4
  • zoneinfo / pytz — conversión de timezone a hora local real
  • APIs: ipwho.is, ipify.org, macvendors.com

Extras del proyecto

  • Historial automático en JSON por módulo y por día en results/
  • 4 modos de color por daltonismo: normal, protanopia, deuteranopia, tritanopia
  • Decorador @is_option que muestra el banner antes de cada módulo
  • Bucle principal en vez de recursión para evitar RecursionError en sesiones largas
  • Banner ASCII de Shockwave generado con arte propio

Instalación y uso

  • Requiere Python 3.9+ (por zoneinfo; Python 3.8 también funciona con pytz instalado)
  • Instalar dependencias: pip install requests phonenumbers
  • Ejecutar: python ShockwaveTR.py
  • Selecciona el módulo con el número del menú · Los resultados se guardan automáticamente en results/
