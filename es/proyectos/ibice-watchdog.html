IBICE Watchdog

Herramienta Python para monitorizar tu red local en tiempo real. Escanea la red mediante ARP, detecta todos los dispositivos conectados con su IP, MAC y fabricante, escanea puertos, clasifica los dispositivos (router, móvil, TV, cámara…) y alerta cuando aparece un intruso. Guarda historial en CSV y muestra un mapa de red.

Activo Python 3 ARP Port Scan MAC Vendor API CSV Windows / Linux

Modos de uso

  • Watchdog — monitorización continua. Escanea cada X segundos y detecta cambios en tiempo real: nuevos dispositivos, desconexiones y cambios de estado.
  • Escáner profundo — análisis completo. Escanea todos los dispositivos y sus puertos abiertos en una sola pasada.

Clasificación de dispositivos

  • (TD) Tu dispositivo
  • (R)  Router de la red
  • (P)  Dispositivo permitido
  • (N)  Dispositivo nuevo
  • (NS) Nuevo sospechoso
  • (S)  Sospechoso confirmado

Cómo detecta dispositivos

  • Ping a toda la red (192.168.x.1–254)
  • Obtiene IP y MAC con arp -a
  • Consulta fabricante vía api.macvendors.com
  • Escanea puertos: 21, 22, 80, 443, 8080, 5000, 9100…
  • Clasifica por MAC, puertos y fabricante

Persistencia y archivos

  • permitidos.txt — lista de dispositivos de confianza
  • historial_ibice.csv — registro completo de actividad con timestamps
  • Mapa de red visual en terminal al finalizar el escaneo

Instalación y uso

  • Requiere Python 3 y pip install requests
  • Ejecutar: python Ibice.py (requiere permisos de admin/root para ARP)
  • También disponible como .exe para Windows — sin necesidad de instalar Python
  • Compatible con Windows (optimizado) y Linux
